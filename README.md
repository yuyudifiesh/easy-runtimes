- 运行脚本提示 "需要提升权限" ？
  - 脚本已经内置提权代码，若不成功请尝试关闭运行窗口并右键文件，选择 "以管理员身份运行" 。
- Git 安装脚本能干什么？
  - 重新安装（默认重装 2.42.0 版本）
  - 配置 git ：设置用户名和邮箱、设置默认分支为 `main` 、设置 Git 使用 LF 换行符、生成 SSH 密钥、添加 SSH 代理。
